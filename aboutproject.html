About the Project

AOT Technologies started the formsflow.ai project to give end users the ability to fill in a form to be processed. The software can also report on form metrics and data.

Some use cases of formsflow are:

  • Licence applications
  • Public submissions
  • FOI requests
  • Funding applications
  • Statements of compliance
  • Employee onboarding
  • Performance reviews
  • Emergency processes
  • Escalations
  • Surveys

  • Case management
