How To Use Formsflow.ai



Designing a Form

User in a designer role can have the following options.

  • Begin the form-design process
  • Specify and customize form elements
  • Edit validations and coding
  • Associate default workflows/deploy a new workflow to a form
  • Publish your form for clients
  • Upload or download a form
  • Admin tab to authorize dashboard to use group

Submitting a form

Client user can

  • View and submit a form
  • Submit and later review an application.
  • Export a a copy of the completed form
  • View the progress of an application

Reviews and approvals

User with Reviewer role can

  • Log in and see pending tasks
  • Claim a particular form
  • Comment on a form
  • Chooses an application to approve
  • Can complete the form, perhaps with feedback
  • Can look at qualitative and quantitative data about form submissions
